1

→ lebih tepat: 6p + 3e = 34000 ...(ii)
3
Dari (i): e = 18500 − 3p, substitusi ke (ii):
6p + 3(18500−3p) = 34000 → 6p + 55500 − 9p = 34000 → −3p = −21500 → p = 7167 (≈ Rp7.000)
4
e = 18500 − 3(7000) = 18500 − 21000 → cek ulang dengan angka bulat: p=5000, b=7000, e=6000

Coba: p=4000, b=6000, e=7500
Ani: 30000+4000+15000=49000 ≠ 47000
Coba: p=5000, b=7000, e=3000
Ani: 35000+5000+6000=46000 ≠ 47000
Coba p=4000, b=6000, e=5500
Ani: 30000+4000+11000=45000≠47000
→ Gunakan eliminasi bersih:
5
Misalkan b=buku, p=pensil, e=penghapus. b=p+2000.
Pers 1: 5b+p+2e=47000 → 5(p+2000)+p+2e=47000 → 6p+2e=37000 →(×3) 18p+6e=111000

No. 32 — Komposisi & Invers Fungsi (Lengkap)
f(x) = 2x−5 dan g(x) = (3x+4)/(2−x), x≠2
A. (f∘g)(x)
= f(g(x)) = 2·(3x+4)/(2−x) − 5 = (6x+8)/(2−x) − 5 = (6x+8 − 5(2−x))/(2−x) = (6x+8−10+5x)/(2−x) = (11x−2)/(2−x)
B. (g∘f)(x)
= g(f(x)) = g(2x−5) = (3(2x−5)+4)/(2−(2x−5)) = (6x−15+4)/(2−2x+5) = (6x−11)/(7−2x)
C. (f∘g)(1)
= (11(1)−2)/(2−1) = 9/1 = 9
D. f⁻¹(x)
y = 2x−5 → x = (y+5)/2 → f⁻¹(x) = (x+5)/2
E. (f∘g)⁻¹(x)
Dari A: y = (11x−2)/(2−x) → y(2−x) = 11x−2 → 2y−xy = 11x−2 → 2y+2 = 11x+xy → 2(y+1) = x(11+y)
→ (f∘g)⁻¹(x) = 2(x+1)/(11+x)
✅ Lengkap seperti di atas
